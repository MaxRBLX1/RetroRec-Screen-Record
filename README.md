## What is RetroRec?

RetroRec is a **lightweight screen recorder** that works on ANY PC — from old dual-core laptops to modern gaming rigs. It uses **DDAGrab GPU capture** and **smart encoding** to record 60 FPS video without lag spikes or performance drops.
