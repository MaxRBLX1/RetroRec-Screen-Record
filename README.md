## What is RetroRec?

RetroRec is a **lightweight, open-source screen recorder** that works on ANY PC — from old dual-core laptops to modern gaming rigs. It wraps FFmpeg's powerful DDAGrab capture engine in a simple, ghost-like interface that never impacts your game performance.

## Quick Start

1. Download `RetroRec.exe` 
2. Place `ffmpeg.exe` in the same folder
3. Run `RetroRec.exe`
4. Press **F12** to start/stop recording
5. Find your videos in `Videos\RetroRec\`
